# SWE370 Data Mining Project

This repository holds the complete source code and evaluation report for the
SWE370 Data Mining Project.

## How to Set Up and Run

```sh
# clone the repo and change directory
git clone git@github.com:mjankari/swe370-data-mining-project.git
cd swe370-data-mining-project

# sync dependencies and run the notebook
uv sync && uv run jupyter lab
```
